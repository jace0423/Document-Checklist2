# Document-Checklist2

課程內容專業驗收表 - FA 審核系統

此專案包含一個用於課程內容專業驗收的 FA (Faculty Advisor) 審核表單。

## 功能特色

- **PM 資料匯入**：支援從 JSON 檔案匯入 PM (專案經理) 提供的資料
- **專業內容檢核**：包含 8 項專業審核標準（知識正確性、邏輯合理性、數據時效性等）
- **簽署功能**：檢核人員與主管簽署區塊，可動態新增/刪除
- **資料匯出**：支援匯出完整 JSON 資料或轉換為 Word 文件
- **列印支援**：優化的列印樣式，適合輸出 PDF

## 檔案結構

- `index.html`: 專案主頁，歡迎頁面與功能介紹
- `fa_review_form.html`: 主要的 HTML 表單檔案，包含所有功能與樣式
- `README.md`: 專案說明文件

## 使用方法

1. **開啟主頁**：在網頁瀏覽器中開啟 `index.html` 或直接訪問根目錄
2. **開始填寫**：點擊「開始填寫表單」按鈕進入 FA 審核表單
3. **匯入 PM 資料**：點擊「📤 匯入 PM 資料」按鈕，選擇 PM 提供的 JSON 檔案
4. **填寫審核內容**：根據 8 項檢核項目進行評估，選擇通過/不通過並填寫修正建議
5. **簽署**：在檢核簽署與主管簽署區塊填寫相關資訊
6. **匯出資料**：
   - 「💾 匯出完整 JSON」：匯出包含所有資料的 JSON 檔案
   - 「📄 匯出為 Word」：轉換為 Word 文件格式
   - 「🖨️ 列印 / 存為 PDF」：直接列印或儲存為 PDF

## 技術需求

- 現代網頁瀏覽器（Chrome、Firefox、Safari、Edge）
- JavaScript 啟用
- 檔案上傳功能（用於匯入 JSON）

## 審核項目說明

1. **知識正確性**：課程觀念、方法、步驟是否正確
2. **邏輯與推論合理性**：因果關係與分析邏輯是否清楚
3. **數據與案例時效性**：使用有效且時效性資料
4. **用詞與術語專業度**：專有名詞使用一致準確
5. **課綱對應度**：內容是否回應教學目標
6. **舉例與情境設計**：範例貼近實際工作場景
7. **工具與版本正確性**：示範最新可用版本工具
8. **內容深度與適切性**：深度符合平台定位

## 注意事項

- 表單資料僅儲存在瀏覽器記憶體中，重新整理頁面將遺失資料
- 建議在填寫過程中定期匯出 JSON 備份
- 匯入的 PM JSON 檔案格式必須正確，否則會顯示錯誤訊息
